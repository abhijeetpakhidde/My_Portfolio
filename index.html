My Projects
Analys of Chest Radiograph in Medical Imaging Using Deep Learning
This project focuses on developing a deep learning–based system to analyze chest radiographs for detecting abnormalities such as cardiomegaly and abnormal lung density. Convolutional Neural Networks (CNNs) are used for automated feature extraction and image classification to accurately identify pathological patterns in X-ray images. The workflow includes image preprocessing, data augmentation, and model optimization to improve performance and generalization. The model is trained and evaluated using labeled chest X-ray datasets with standard metrics such as accuracy, precision, and recall. The goal of the project is to support radiologists by providing reliable, fast, and consistent preliminary diagnostic insights, ultimately improving diagnostic accuracy and efficiency.
